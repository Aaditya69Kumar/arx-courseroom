#  arX CourseRoom

A role-based smart classroom platform designed to streamline digital education within institutions.

---

##  Overview

**arX CourseRoom** is a full-stack Learning Management System (LMS) that enables seamless interaction between students and instructors. It eliminates dependency on third-party platforms by providing a customizable, institution-focused solution with centralized academic management.

---

##  Problem Statement

* Lack of unified digital classroom systems in institutions
* Dependency on external platforms (data privacy & control issues)
* Poor management of assignments, deadlines, and communication
* Limited visibility for administrators

---

##  Solution

arX CourseRoom provides a **role-based platform** with dedicated dashboards for:

*  Instructor
*  Student
*  Admin *(in progress)*

---

##  Features

###  Instructor

* Create & manage courses
* Upload notes, assignments, quizzes, PYQs (PDF)
* Post announcements
* Grade assignments
* Edit/Delete all content
* Chat with students

###  Student

* Enroll in courses
* View materials & announcements
* Submit assignments
* Track deadlines
* Search instructors
* Chat with teachers

###  Admin (Planned)

* Monitor all users & courses
* Track submissions
* Manage platform access

---

##  Tech Stack

| Layer    | Technology                        |
| -------- | --------------------------------- |
| Frontend | Next.js, TypeScript, Tailwind CSS |
| Backend  | Next.js API Routes                |
| Auth     | NextAuth                          |
| Database | MySql                             |
| Storage  | Cloudinary                        |

---

##  Architecture

* Client Layer → Dashboards & UI
* API Layer → Authentication, role-based routing
* Data Layer → Database & storage

---

---

##  How It Works

1. User signs up / logs in
2. Select role (Instructor / Student)
3. Access dedicated dashboard
4. Perform role-specific actions
